# Lantern Drift Atlas

> **Why.** Interactive canvas exploring a sky of drifting lanterns with metadata trails and temporal filtering.

## Features

- Animated canvas sky with 80+ drifting lanterns carrying origin/wish/heat metadata
- Click a lantern to pin its trail and inspect its lineage panel
- Time-slider scrubs release windows with live heat-map overlay
