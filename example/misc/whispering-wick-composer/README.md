# Whispering Wick Composer

> **Why.** Craft bespoke paper lanterns by composing shape, glow, and message — then export the recipe.

## Features

- Live SVG preview that re-renders as you tune color, shape, pattern, and flicker
- Materials calculator estimating paper, wire, and burn time from chosen dimensions
- Copy-ready recipe card with JSON export and random inspiration shuffle
