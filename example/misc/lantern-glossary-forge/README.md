# Lantern Glossary Forge

> **Why.** Composes ceremonial lantern inscriptions and distills a ubiquitous-language glossary from the drafting session.

## Features

- Inscription composer with tone/occasion/element knobs and live calligraphic preview
- Auto-extracted ubiquitous-language glossary of recurring terms with canonical forms
- Export as JSON; copy a single inscription; seed 8 preset occasions

## File structure

```
lantern-glossary-forge/
  index.html    — shell, markup, inline SVG where used
  style.css     — dark-theme styling and animations
  app.js        — interactions, simulated data, render loop
  manifest.json — hub metadata
```

## Usage

```bash
# any static file server works
python -m http.server 8080
# or open index.html directly in a browser
```
