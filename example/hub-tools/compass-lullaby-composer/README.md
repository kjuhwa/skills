# Compass Lullaby Composer

> **Why.** A practical composer that turns brass compass bearings into hummed notes, capturing each choice as an immutable musical act.

## Features

- Dial built around lantern-visualization-pattern glow — the needle halo brightens with each added note
- Every added/undone note flows through an immutable-action-event-log, replayable into a full score
- Duration selector follows period-mode-enum-config — one enum maps symbols to seconds, note beats, and export text
