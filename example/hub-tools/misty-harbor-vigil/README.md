# Misty Harbor Lantern Vigil

> **Why.** Release paper lanterns into the drifting wind above sleeping fishing boats and guide them past the constellations.

## Features

- Click-to-release paper lanterns with physics, heat lift, and stochastic drift
- Wind dial and temperature slider shift lantern trajectories in real time
- Score vigil — count lanterns that reach the constellations vs. snuff in the mist
